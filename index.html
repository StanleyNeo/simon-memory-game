How to Play

  1. Watch the sequence of colors that Simon shows
  2. Repeat the sequence by clicking the colors in the same order
  3. The sequence gets longer each round
  4. If you make a mistake, the game is over
  5. Try to reach the highest level possible!

You can also press any key to start the game.
